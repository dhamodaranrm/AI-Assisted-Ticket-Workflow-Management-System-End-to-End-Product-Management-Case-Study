# User Stories

## User Story 1
As a support engineer,
I want tickets to be automatically categorized,
So that I can reduce manual effort.

### Acceptance Criteria
- AI categorizes tickets automatically
- Categories are editable
- Priority is suggested

---

## User Story 2
As a manager,
I want SLA breach predictions,
So that I can proactively manage risks.

### Acceptance Criteria
- Dashboard shows risk indicators
- Alerts are generated before SLA breach
